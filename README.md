## Projects

### 📬 Email Scraper — Metis, IIT Gandhinagar
> Automated email pipeline built for the campus insIIT platform

Built the **entire system end-to-end** as part of the Metis club. The pipeline polls Gmail every 5 minutes via the Gmail API, runs emails through a rule-based pre-classifier, parses content, classifies into categories (mess menu, medical updates, events, opportunities, timetables), and writes structured data to a database.

**What I built:** Gmail API integration with OAuth2, the full pipeline orchestration, inter-module communication, and the scheduler loop.

**Stack:** Python · Gmail API · OAuth2 · APScheduler · MongoDB  
**Repo:** [yuvraj-rathod-1202/Email-Scrapper](https://github.com/yuvraj-rathod-1202/Email-Scrapper) *(club repo — PRs under my account)*
